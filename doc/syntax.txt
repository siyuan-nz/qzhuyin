% comment

\settopmargin{mm}
Set the top page margin in millimeters.

\setbottommargin{mm}
Set the bottom page margin in millimeters.

\setleftmargin{mm}
Set the left page margin in millimeters.

\setrightmargin{mm}
Set the right page margin in millimeters.

\setfont{font}
Specify a font to use.

\setfontsize{size pt}
Specify the size of the font in points.

\hspace{characters}
Fill with specified number of characters of horizontal space.

\vspace{characters}
Fill with specified number of characters of vertical space.

\zhuyin{bopomofo}
Add zhu yin to the sentence delimited by space.

\newpage{}
Start a new page

\\
Line feed.

Blank line(s)
Change paragraph.

\label{name}
Create a label, optionally with a name, which can be referred to later. See
\ref{}.

\ref{name}
Refer to a previously defined label, optionally by name. If a name is not
specified, the reference will be matched with the first unreferenced \label.
